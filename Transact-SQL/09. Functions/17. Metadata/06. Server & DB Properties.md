## **2. DATABASEPROPERTYEX()**
- **Purpose:** Returns properties of a database.
- **Syntax:**
  ```sql

  ```
- **Common Properties:**
  - `'Collation'` â†’ Returns the collation setting of the database.
  - `'Recovery'` â†’ Returns the recovery model (`FULL`, `SIMPLE`, `BULK_LOGGED`).
  - `'Status'` â†’ Returns the current status (`ONLINE`, `OFFLINE`, `RESTORING`).

