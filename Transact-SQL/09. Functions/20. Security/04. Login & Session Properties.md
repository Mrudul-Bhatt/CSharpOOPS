## **1. LOGINPROPERTY()**
- **Purpose:** Returns properties of a SQL Server login, such as password expiration, lockout status, and failed login attempts.
- **Syntax:**
  ```sql

  ```
- **Common Properties:**
  - `'BadPasswordCount'` â†’ Number of consecutive failed login attempts.
  - `'BadPasswordTime'` â†’ Timestamp of the last failed login attempt.
  - `'DaysUntilExpiration'` â†’ Number of days until the password expires.
  - `'IsExpired'` â†’ Returns `1` if the password has expired.
  - `'IsLocked'` â†’ Returns `1` if the login is locked.
  - `'PasswordLastSetTime'` â†’ Timestamp of the last password change.
- **Example Usage:**
  ```sql
  SELECT LOGINPROPERTY('sa', 'BadPasswordCount') AS FailedAttempts,
         LOGINPROPERTY('sa', 'IsLocked') AS IsLocked,
         LOGINPROPERTY('sa', 'DaysUntilExpiration') AS DaysLeft;
  ```
  **Output Example:**  
  ```
  FailedAttempts | IsLocked | DaysLeft
  -----------------------------------
  3             | 0        | 10
  ```
- **Use Case:** Helps in **monitoring login security** and **password expiration policies**.

## **2. SESSIONPROPERTY()**
- **Purpose:** Returns properties of the current session, such as ANSI settings and transaction isolation level.
- **Syntax:**
  ```sql

  ```
- **Common Properties:**
  - `'ANSI_NULLS'` â†’ Returns `1` if ANSI NULLS is enabled.
  - `'ANSI_WARNINGS'` â†’ Returns `1` if ANSI warnings are enabled.
  - `'TRANSACTION_ISOLATION_LEVEL'` â†’ Returns the current transaction isolation level.
- **Example Usage:**

  **Output Example:**

  ```
  --------------
  ```

- **Use Case:** Helps in **checking session settings** before executing queries.
